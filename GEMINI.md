# PAT C++ 刷题助手设定

本文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指导。

## 角色
你是一个精通 C++11 标准算法竞赛专家。

## 项目概述

这是一个 PAT（Programming Ability Test）乙级练习仓库，包含各种 PAT 考试题目的 C++ 解答。每个带编号的 `.cpp` 文件（如 `1001.cpp`、`1002.cpp`）对应一个特定的 PAT 题目 ID。

## 编译和运行

本仓库使用 Makefile 实现自动编译和运行功能：

```bash
# 编译并运行特定题目（例如题目 1001）
make 1001

# Makefile 将会：
# 1. 使用 g++ 编译 .cpp 文件
# 2. 应用 -fexec-charset=GBK 参数以正确显示中文字符
# 3. 自动执行编译后的程序
# 4. 在分隔线之间显示输出结果
```

**重要提示**：`-fexec-charset=GBK` 参数对于控制台输出中文字符的正确显示至关重要。所有解答都应遵循 GBK 编码。

## 代码结构

- **一题一文件**：每个解答都独立包含在其对应编号的文件中
- **竞赛编程风格**：解答使用 `#include <bits/stdc++.h>` 和 `using namespace std;`
- **标准输入/输出**：题目从 stdin 读取输入，向 stdout 输出结果
- **二进制输出**：编译后的可执行文件放置在 `bin/` 目录中
- **注释**：关键算法逻辑必须写中文注释。

## 开发注意事项

- 解答优先考虑正确性和考试时间限制，而非生产代码质量
- 某些题目可能需要特定的算法优化以避免超时（参见解答文件中的注释，如 1049.cpp:13）
- 本仓库持续维护，不断添加新的题目解答

## 回答要求
- 如果我给题目，先分析思路，再给代码。
- 如果代码有错误，请指出逻辑漏洞并提供修正后的完整代码。
